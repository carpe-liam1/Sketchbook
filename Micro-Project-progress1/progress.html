Progress #1

Summary

The primary function of my sketch is to create an interactive experience which will allow the user to bend and warp the perimeter of the ellipse put on screen. Through use of different keys and mouse movement, the user will be able to "push" or "pull" specific points on the perimeter of the ellipse, manipulating its overall shape. This change in shape will be the main variable changing over time. However, I also plan to create a function which will allow the user to also cause the ellipse to revert back gradually to its ellipse form, dispaying the change in form as a fluid process as well.

Primary variable

The primary variable in this sketch is the distance between the position of MouseX and MouseY and the nearest point on the perimeter of the ellipse. This initial distance (stage=0) will be added to or subtracted from by manipulation of the ellipse and its shape (Stage++ or Stage--).

Base Build

This week I plan to code the following: - The primary variable which I will designate as Distortion[] - An input which will be the mousePressed function making and if statement where if mousePressed happens then the nearest point of the ellipse perimeter will beginning to be pulled towards the mouse position, thus warping the shape. - The first state will be the original ellipse with Distortion[0] applied. The second state which will be some kind of warp effect applied to the ellipse with Distortion[i] applied.
