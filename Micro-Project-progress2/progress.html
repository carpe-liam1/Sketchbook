Pressure

The pressure in my project comes from the additional lerp function implemented as an else feature. These action draws the points which have been altered back to their original positon if the mouse is not presses within their vicinity.

Structural Addition

This week I added a pause featue which counteracts the pressure lerp I've mentioned above. With the pause button, I made the function of mousePressed part of a larger true or false variable which freezes the movement of points. This way, the user can pause the distortion they've done and analyze it without It changing shape. Additionally, this feature allows for the user to mess with the legibility of the text by pausing and unpausing the movement until they are satisfied with the result. Two things I want to do is rewrite the code so that the pause button is instead an unpause button so that the movement of the lerp function isn't constantly affecting the text as well as Another Idea I have would be to reverse the movement which has happened with points, however I'm not informed enough to understand what kind of code this would entail.
