Interactive Sketchbook - Sketch 9

for this sketch, I built upon my previous sketch of images changes in accordance with the movement of the mouse. This time, I decided to incorporate a new feature which was the ability for the user to reverse image order. THis is tracked by which direction the users mouse goes in. If they move to the right, the images will advance forward, and if they move to the left, the images will go backward. going through this showed me the importance of understnad block logic. WHen I originially set the code for this sketch, I found that it wasn't working because i had nested the left-direction movement inside the right direction movement, thus never allowing for it to occur. Additionally, I added text to represent which image was being displayed which allows the user to better understand what they are seeing
